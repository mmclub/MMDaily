# MMDialy

南邮移动互联网俱乐部第一次内部迷你Hackthon项目

## Objective

- 完成一款Andorid平台的日报类应用，参考知乎日报
- 内容与应用逻辑解除耦合，完成应用后可以为不同的内容平台适配，编译成多个应用（比如适配基于Wordpress的JSON-API插件的API，和自己写的Web后台服务
